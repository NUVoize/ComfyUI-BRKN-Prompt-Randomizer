# BRKN Thumbnail Capture Pack

This is a constrained utility pack for consistent BRKN catalog thumbnails.

It is not a normal lifestyle prompt pack and it does not load Abby's LoRA. The
Abby LoRA filename, trigger token, and default strength are intentionally left
pending in `manifest.json` until confirmed.

The randomizer should treat a selected capture recipe as a high-priority
composition constraint. The browser may discover and select this pack, but it
must not render thumbnail prompts itself.
